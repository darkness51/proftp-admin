Django Proftp Users Admin
=========================

Administrador de usuarios para Proftp escrita en django.

Puede ser usado con cualquier base de datos soportada por proftp. De momento, permite lo siguiente:

- Agregar usuarios.
- Modificar usuarios.
- Eliminar usuarios.
- Crear cuotas de disco.
